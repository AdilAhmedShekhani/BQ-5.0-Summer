# Web Development with AI 🚀

## Course Outline

### **1. Introduction to Web Development & AI-Assisted Coding**

-   What is web development and how the internet works
-   Role of HTML, CSS, and JavaScript
-   Introduction to AI tools (ChatGPT, Gemini) for learning and coding
-   Setting up the environment (VS Code, browser)
-   How to use AI to ask questions and get code help
-   **Hands-on:** Create your very first HTML page with AI assistance

### **2. Prompt Engineering + HTML Basics**

-   What is prompt engineering and why it helps beginners
-   Writing simple, clear prompts for ChatGPT/Gemini
-   Using AI to generate and explain HTML code
-   Essential HTML elements: headings, paragraphs, lists, links, images
-   Building a basic webpage structure
-   **Hands-on:** Build a simple personal bio page using AI-generated
    HTML

### **3. CSS Fundamentals -- Styling Your Website**

-   What is CSS and how it connects to HTML
-   CSS selectors, properties, and the box model
-   Colors, fonts, spacing, and borders
-   Introduction to Flexbox for layout
-   Using AI to generate and improve CSS styles
-   **Hands-on:** Style your bio page with colors, fonts, and layouts

### **4. Responsive Design & Git Basics**

-   Making websites responsive for mobile and desktop
-   Media queries and mobile-first design
-   Basic CSS animations and transitions
-   Introduction to Git and version control
-   Git basics: init, add, commit, push
-   Creating a GitHub account and uploading projects
-   **Hands-on:** Make your website responsive and push it to GitHub

### **5. Logic Building & JavaScript Introduction**

-   Programming logic and problem-solving
-   Variables, data types, and operators
-   Conditional statements (if/else)
-   Loops (for, while)
-   JavaScript syntax and browser console
-   **Hands-on:** Solve simple logic problems and write your first
    JavaScript

### **6. JavaScript Fundamentals -- Functions & DOM**

-   Functions
-   Understanding the DOM
-   Selecting and updating HTML elements
-   Click and input events
-   **Hands-on:** Build a button counter or color changer

### **7. JavaScript -- Building Interactive Pages**

-   Working with forms
-   Arrays and objects
-   Showing and hiding elements
-   Using AI to write and debug JavaScript
-   **Hands-on:** Build a To-Do List or Quiz App

### **8. Final Project + Deployment**

-   Plan and build a project with AI
-   Portfolio, Landing Page, or Quiz App
-   Styling, responsiveness, and interactivity
-   Deploy to GitHub Pages
-   **Hands-on:** Present your final project

## 🎯 Learning Outcome

After completing this course, students will be able to: - Build modern
responsive websites. - Use AI tools effectively for coding. - Write
HTML, CSS, and JavaScript confidently. - Upload projects to GitHub. -
Deploy websites using GitHub Pages. - Build a complete frontend project
independently.
