# Web Development with AI 🚀

# Class 1 -- Course Introduction

Welcome to **Web Development with AI**.

This course is designed for complete beginners. If you have never
written a single line of code before, don't worry. We will learn
everything step by step using simple English, practical examples, and AI
tools like ChatGPT.

Our goal is not only to learn coding but also to learn **how to use AI
as a smart coding assistant**.

------------------------------------------------------------------------

# What is Web Development?

Web Development is the process of creating websites and web
applications.

Every website is made with three main technologies:

-   **HTML** -- Creates the structure of a webpage.
-   **CSS** -- Makes the webpage beautiful.
-   **JavaScript** -- Makes the webpage interactive.

During this course, we will learn all three from the beginning.
